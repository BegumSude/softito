# Softito Python Başlangıç ve Temel Uygulamaları

Bu depo (repository), Softito kapsamında gerçekleştirilen temel Python programlama dersleri, başlangıç seviyesi uygulamalar, sınıflar (classes) ile ilgili pratik sorular ve temel seviye değerlendirme quizlerini içermektedir.

## Proje İçeriği

Bu projede yer alan Jupyter Notebook dosyaları ve içerikleri aşağıda listelenmiştir:

*   **`python_baslangic.ipynb`**: Python programlama diline giriş, temel syntax, değişkenler ve veri tipleri.
*   **`temel_python.ipynb`**: Kontrol yapıları (if-else), döngüler (for, while) ve temel fonksiyonlar.
*   **`temel_python_2.ipynb`**: Listeler, demetler (tuples), sözlükler (dictionaries) ve kümeler (sets) gibi veri yapıları.
*   **`python_temel_quiz.ipynb`**: Temel Python konularını pekiştirmek amacıyla hazırlanmış pratik test ve sorular.
*   **`python_class_sorular.ipynb`**: Nesne Yönelimli Programlama (OOP) mantığı, sınıflar (classes), nesneler (objects), metotlar ve kalıtım konularına yönelik uygulamalı soru ve çözümler.

## Başlangıç

Bu notebook dosyalarını kendi bilgisayarınızda çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

1.  **Depoyu Klonlayın:**
    ```bash
    git clone https://github.com/BegumSude/softito.git
    cd softito
    ```

2.  **Gereksinimleri Kurun:**
    Jupyter Notebook veya JupyterLab kurulu değilse kurun:
    ```bash
    pip install jupyter
    ```

3.  **Notebook'ları Başlatın:**
    ```bash
    jupyter notebook
    ```
